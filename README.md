### API setup

- Clone backend repo:

```
git clone https://github.com/DavidCheungTW/floating-books-api.git
```

- Move into the directory and install:

```
cd floating-books-api
npm install
```

- Create .env files:

```
PGPASSWORD=password
PGDATABASE=floating_books_api_dev
PGUSER=postgres
PGHOST=localhost
PGPORT=5432
PORT=3000
```

- To start and run on port 3000:

```
npm start
```
